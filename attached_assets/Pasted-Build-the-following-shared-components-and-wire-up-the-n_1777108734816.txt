--- components/BottomTabBar.js ---
A custom bottom tab bar component (not using React Navigation's default tab bar —
we want full control over styling).

Props: activeTab (string), onTabChange (function).
Tabs: { id: 'map', label: 'Map', icon: '🗺️' }, { id: 'jobs', label: 'Jobs', icon: '📋' },
      { id: 'earnings', label: 'Earnings', icon: '💰' }, { id: 'profile', label: 'Profile', icon: '👤' }

Styling: white background, borderTopWidth 1, borderTopColor #F3F4F6, paddingBottom 8
(safe area). Each tab is a TouchableOpacity with flex 1, alignItems center. Active tab:
icon + label in #1E9B6B, fontWeight 600. Inactive: #9CA3AF.

--- navigation/CollectorNavigator.js ---
A React Navigation Stack + custom tab implementation:
- Stack: Login → Main (tab container)
- The "Main" screen is a custom tab container that renders the correct screen based
  on the activeTab state, with BottomTabBar at the bottom.
- Tab screens: HotspotMapScreen (map), JobListScreen (jobs),
  EarningsDashboardScreen (earnings), a simple ProfileScreen placeholder
  (just show collector name, city, verified badge, and a logout button).
- The Stack handles NavigationScreen, JobDetailScreen, WeightEntryScreen,
  PickupConfirmationScreen as modals pushed on top.
